# XAU_M1_Model_10_BullsBears_Force_EA.ex5 使用说明

## 功能
黄金 M1 多场景指标模型 EA，使用 Bulls Power、Bears Power、Force Index 与 EMA 方向过滤，自动识别上行、下行和突破场景。

## 出场
不使用 ATR 和固定止盈，默认采用阶梯移动止盈、反向信号平仓、结构保护、最大浮亏保护和持仓时间保护。

## 回测参考
2026.05.26 至 2026.06.25，XAUUSDm M1，点差 360，净利润 48.11，最大净值回撤 0.14%，胜率 78.85%，夏普 75.70。

## 风险
历史测试不代表未来收益。实盘前必须在相同品种后缀、点差和账户条件下复测。
