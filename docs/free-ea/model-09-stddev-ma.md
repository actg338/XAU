# XAU_M1_Model_09_StdDev_MA_Channel_EA.ex5 使用说明

## 功能
黄金 M1 StdDev + EMA 通道模型，通过标准差离散度判断突破、趋势延续和震荡回归场景。

## 出场
不使用 ATR 和固定止盈，默认采用阶梯移动止盈、反向信号平仓、结构保护、最大浮亏保护和持仓时间保护。

## 回测参考
2026.05.26 至 2026.06.25，XAUUSDm M1，点差 360，净利润 35.50，最大净值回撤 0.23%，胜率 68.27%，夏普 53.00。

## 风险
标准差模型在极端新闻行情中可能出现快速反复信号，建议先用模拟账户验证。
