# XAU_M1_Model_03_BB_Stochastic_EA.ex5 使用说明

## 功能
黄金 M1 Bollinger 带宽 + Stochastic 模型，结合布林带突破、中轴方向和随机指标动量判断多场景信号。

## 出场
不使用 ATR 和固定止盈，默认采用阶梯移动止盈、反向信号平仓、结构保护、最大浮亏保护和持仓时间保护。

## 回测参考
2026.05.26 至 2026.06.25，XAUUSDm M1，点差 360，净利润 27.59，最大净值回撤 0.38%，胜率 70.19%，夏普 40.90。

## 风险
布林与 Stochastic 在横盘中信号较多，应保持日内交易次数限制和冷却机制。
