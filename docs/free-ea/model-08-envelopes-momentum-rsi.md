# XAU_M1_Model_08_Envelopes_Momentum_RSI_EA.ex5 使用说明

## 功能
黄金 M1 Envelopes + Momentum + RSI 模型，用包络线位置、动量和 RSI 过滤趋势、震荡和突破场景。

## 出场
不使用 ATR 和固定止盈，默认采用阶梯移动止盈、反向信号平仓、结构保护、最大浮亏保护和持仓时间保护。

## 回测参考
2026.05.26 至 2026.06.25，XAUUSDm M1，点差 360，净利润 27.97，最大净值回撤 0.21%，胜率 75.00%，夏普 40.27。

## 风险
包络线参数对不同报价源敏感，建议根据经纪商点差和报价密度复测。
