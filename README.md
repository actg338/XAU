# XAU策略量化交易

专业的黄金量化交易解决方案网站

## 项目简介

这是一个展示XAU（黄金）量化交易策略的静态网站。网站采用现代化的设计风格，以金色和深色为主题色，体现黄金交易的专业性。

## 功能特点

- 🤖 智能算法 - 采用先进的机器学习算法
- 📈 实时分析 - 24/7全天候监控市场数据
- 🛡️ 风险管理 - 内置智能风控系统
- ⚡ 高速执行 - 毫秒级订单执行速度
- 📊 数据可视化 - 直观的图表展示
- 🔔 实时提醒 - 重要信号即时推送

## 本地运行

使用Python启动本地服务器：

```bash
python3 -m http.server 8080
```

然后在浏览器中访问 `http://localhost:8080`

## 技术栈

- HTML5
- CSS3 (响应式设计)
- JavaScript (原生)
- Canvas API (价格图表)
